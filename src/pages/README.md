# HappyBride

HappyBride is a wedding planning React application created with TypeScript.

The application helps users organize:
- wedding guests
- budget planning
- vendors
- inspiration gallery
- tasks and planning progress

Technologies used:
- React
- TypeScript
- CSS
- React Router
- Git & GitHub

Features:
- reusable components
- pagination
- filtering
- API integration
- localStorage persistence
- dynamic routing
